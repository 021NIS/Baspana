# Baspana

## Getting Started

1.  Clone the repo

```bash
$ git clone git@github.com:021NIS/Kundelik.git
$ cd Kundelik
$ rm -rf .git
$ git init
```

2.  Install dependencies from [npm](https://www.npmjs.com)

```bash
$ npm install
$ react-native run-ios (or run-android)
```
